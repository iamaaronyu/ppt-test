03 · 10 / 13
Troubleshooting

常见问题

Problem 01

sessions should be nested with care

执行 tmux attach -t 3 时报错。原因：当前已在 Tmux 中，不能嵌套 attach。

✅ 解决方案
tmux switch-client -t 3
或: Ctrl+B, S 选择目标 Session
Problem 02

no server running

执行 tmux ls 提示无服务。原因：当前没有存活的 Tmux Session。

✅ 解决方案
tmux new -s work
重新创建会话即可
Quick Check
如何判断是否在 Tmux 中？ echo $TMUX 有输出 = 在 Tmux 中
