Act III · 进阶技巧

Beyond the Basics

进阶技巧

掌握基础命令之后，这些进阶技巧能让你的 tmux 体验再上一个台阶。

社区最佳实践

01

插件生态

TPM（tmux Plugin Manager）一键安装插件。tmux-sensible 合理默认值、tmux-resurrect 会话持久化、tmux-continuum 自动保存——社区已经把轮子造好了。

git clone https://github.com/tmux-plugins/tpm

02

鼠标支持

开启鼠标模式——点击切换 Panel、拖拽调整分屏大小、滚轮翻页。终端操作瞬间现代化，告别纯键盘的约束。

set -g mouse on

03

美化与配置

定制状态栏（显示时间/CPU/电池）、更换配色方案、用 tmuxp 写 YAML 配置文件一键启动复杂工作区布局。

tmuxp load my-workspace.yaml
