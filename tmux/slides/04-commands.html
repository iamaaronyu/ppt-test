Act I · 常用命令

Essential Commands

从创建
到恢复

tmux 的命令行设计遵循一致的命名模式。记住两条主干：Session 的生命周期管理，和 Window/Pane 的日常操作。

提示

所有 tmux 命令都支持 tab 补全。输入 tmux 后按 Tab 即可浏览全部子命令。

tmux new -s name

02

分离会话

03

tmux ls

04

重新接入

tmux attach -t name

05

杀掉会话

tmux kill-session
