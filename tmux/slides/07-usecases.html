Real-world Usage

每个人都在
用 tmux 做什么

从个人开发者到运维团队，tmux 的六个经典使用场景——覆盖了日常工作的方方面面。

共通点

所有场景都围绕同一个核心：让终端工作区脱离 SSH 连接的物理限制，成为独立存在的「虚拟桌面」。

远程开发

SSH 到开发机，在 tmux Session 中打开 Vim、数据库客户端、日志监控。网络断了也不怕。

长期监控

htop 看资源、tail -f 追日志、watch 定时刷新——三个 Pane 并排，关掉终端监控继续。

结对编程

两人同时 attach 到同一个 tmux Session，输入同步显示——最轻量的远程协作方案。

部署与运维

生产服务器上开 tmux Session 执行部署。网络抖动不中断——运维基本素养。

工作区持久化

用 tmux 脚本或插件保存完整工作区布局，重启机器后一键恢复所有窗口和分屏。

多任务并行

Window 1 跑测试，Window 2 写代码，Window 3 看文档——Ctrl+B, N 秒切上下文。
